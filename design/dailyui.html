Visual Design, Interaction Design

In an effort to improve my visual design skills and as well as learn some interaction design, I decided to challenge myself by taking on the Daily UI Challenge. Daily UI sends the user a design prompt everyday and it is up to the user to intepret the prompt and solution.

I also decided to make interactions or native prototypes on, hopefully, all of the challenges I take so that I can learn how to use Origami Studio (and possibly other programs) and hone my interaction design skills. This page will be updated every once in a while as I am currently taking on this challenge.

Wish me luck, hopefully I don't die and give up part way! 💪
